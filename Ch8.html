Chapter 8 Terms

Chapter 8 focuses on cybersecurity and protecting information from unauthorized access. Here are the key concepts described in my own words, based on the textbook.

Key Terms
Term Definition
Information SecurityKeeping information protected from people who are not authorized to access it.
MalwareAny malicious software designed to harm or infiltrate a computer.
VirusSoftware that hides inside another program/file and spreads when activated.
WormSelf-replicating software that spreads on its own across networks.
Trojan HorseSoftware that looks useful or harmless but secretly performs harmful actions.
Keystroke LoggerHidden program that records what a user types, including passwords.
SpamUnwanted mass email, often advertising or malicious.
Denial-of-Service Attack (DoS)Overloading a website so real users can’t access it.
BotnetA group of machines infected and remotely controlled to perform attacks.
RansomwareSoftware that locks files and demands money to unlock them.
PhishingTricking people into giving personal info by pretending to be trustworthy.
Antivirus SoftwareScans for and removes malware.
FirewallControls data allowed in or out of a computer or network.
Security PatchesUpdates that fix security weaknesses.
EncryptionScrambling readable data so only someone with a key can understand it.
Caesar CipherSimple shift cipher that moves letters a fixed distance.
Stream CipherEncrypts data one character at a time.
Block CipherEncrypts data in blocks, mixing info across multiple characters.
DESA 56-bit older symmetric block cipher.
Triple DES (3DES)Encrypt-decrypt-encrypt process for stronger security.
AESModern symmetric encryption with 128–256-bit keys.
Public-Key EncryptionUses two keys: one public for locking, one private for unlocking.
RSAPublic-key encryption based on multiplying large prime numbers.
Public-Key CertificateProves that a server is who it claims to be.
HandshakeInitial secure setup before sending protected data.
VPNEncrypted connection that hides your IP and protects your traffic.
RDPLets users control a remote computer’s desktop.
Embedded ComputersSmall computers built into other systems to perform limited tasks.
IoT — Internet of ThingsEveryday devices connected to the Internet to send/receive data.
Mirai BotnetA botnet that hijacks IoT devices (often with default passwords) to launch major attacks.

Brackets — Questions & Answers

  1. What is Brackets?
    Brackets is a free, open-source editor for coding, especially HTML, CSS, and JavaScript. I like that it has Live Preview so I can immediately see changes.
  2. How much does it cost?
    It doesn’t cost anything — it’s completely free.
  3. Several useful extensions:
    • Beautify — cleans up messy code formatting
    • Emmet — speeds up HTML/CSS writing
    • Autoprefixer — helps CSS work on different browsers
    • Markdown Preview — lets you preview Markdown live
    • Theme Packs — lets you tweak the look of the editor

“Discover Crypt” — Games & Reflections

Games I Played & What I Learned

  1. Easy Cipher Game:
    I started with simple substitution ciphers. It helped me understand how predictable letter patterns can make breaking codes easier.
  2. Normal Cipher Game:
    This level made me pay attention to letter frequency and patterns. I started recognizing how certain letters appear more often.
  3. Hard Cipher Game:
    Hard mode was more complicated and showed how block-style encryption can make messages much harder to break since characters influence each other.
