Brackets — Questions & Answers

  1. What is Brackets?
    A free, open‑source code editor focused on HTML, CSS, and JavaScript. It includes features like Live Preview and Quick Edit and is maintained by the open‑source community. (See brackets.io)
  2. How much does it cost?
    Free (open source, MIT‑licensed).
  3. Several useful extensions:
    • Beautify — formats/pretty‑prints code.
    • Emmet — rapid HTML/CSS authoring shortcuts.
    • Autoprefixer — adds vendor prefixes to CSS.
    • Markdown Preview — live preview for Markdown files.
    • Theme packs — alternate editor color themes.

Note: Adobe ended official support in 2021, but Brackets remains available via the community and forks. VS Code is a common alternative with similar extensions.
