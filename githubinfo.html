All About GitHub

GitHub is a web-based platform used for version control and collaborative software development. It allows developers to host and review code, manage projects, and collaborate with others. Here are some key features and information about GitHub:

Q&A:

What is a version control system?
A tool that tracks changes to files so you can go back to older versions and work with others easily.
What is Git?
A free version control system that saves snapshots of your code as you work.
What is GitHub?
A website where you can store Git projects online, share them, and collaborate with others.
What is the difference between Git and GitHub?
Git is the tool that tracks changes on your computer. GitHub is the website that hosts Git projects online.
Who started GitHub and how was it started?
GitHub was founded in 2008 by Tom Preston-Werner, Chris Wanstrath, and PJ Hyett to make sharing Git projects easier.
What company owns it now?
Microsoft owns GitHub (they bought it in 2018).
How much does a GitHub account cost?
GitHub offers free accounts with paid upgrades for extra features.
What is the Octocat?
The Octocat is GitHub’s mascot — a cat with octopus arms used in their logo and designs.

Key Terms:

  1. Repository: A folder where all your project files are stored.
  2. Commit: A saved change or snapshot of your work.
  3. Fork: A copy of someone else's repository that you can edit in your own account.
  4. Push: Sending your saved changes (commits) from your computer to GitHub.
  5. Pull request: A request to add your changes into another branch or repository.
  6. Workflow: The step-by-step process or automation for how code is built, tested, and shared.
  7. Issues: A way to track bugs, tasks, or ideas for a project.
  8. Raw button: Shows the plain text version of a file's code.
  9. Blame button: Shows who made changes to each line of a file.
